Converts pair of associated key and value arrays to equivalent key-value array.

Converts incoming associated arrays formatted as: [key1, key2, key3] [value1, value2, value3]: to single array formatted as: [key1, value1, key2, value2, key3, value3]

Parameters
[in]keysThe array of keys
[in]valuesThe array to values
[out]key_valuesThe flat array of keys and values to be filled by combining 'keys' and 'values'
[in]assoc_lenThe length of the 'keys' and 'values' arrays (should be the same)
[in]kv_lenThe length of the 'key_value' array

Converts key-value array to pair of associated key and value arrays.

Convert incoming array formatted as: [key1, value1, key2, value2, key3, value3] to two associated arrays formatted as: [key1, key2, key3] [value1, value2, value3]

Parameters
[in]key_valuesThe flat array of keys and values
[out]keysThe array to be filled with the keys from 'key_values'
[out]valuesThe array to be filled with the values for 'key_values'
[in]kv_lenThe length of the 'key_value' array
[in]assoc_lenThe length of the 'keys' and 'values' arrays (should be the same)
